Silaty
======

A neat prayer reminder app in GTK

![screenshot](Silaty.png)

## Installation

To install open a terminal & run: 
```bash
cd /path/to/silaty
sudo ./install.sh
```

To uninstall run:
```bash
sudo ./uninstall.sh
```
