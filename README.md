Silaty
======

A neat prayer reminder app in GTK

![screenshot](screenshots/Silaty.png)

## Test

Make sure you have python3 installed using `python3 -V` cmd, then run: 
```bash
cd /path/to/silaty
python3 silaty-indicator.py
```

## Installation

To install open a terminal & run: 
```bash
cd /path/to/silaty
sudo ./install.sh
```

To uninstall run:
```bash
sudo ./uninstall.sh
```
